## Build the Image

```bash
cd iris-ml-pipeline
docker build -t iris-ml-pipeline:latest .
```

**What happens during build:**

| Stage | Action |
|-------|--------|
| `builder` | Installs all Python dependencies from `requirements.txt` |
| `runtime` | Copies only installed packages + `app.py` + `models/` — nothing else |

> ⚡ **Layer caching:** If `requirements.txt` hasn't changed, the dependency install step is skipped on rebuilds.

---

## Run the Container Locally

```bash
docker run -p 8000:8000 iris-ml-pipeline:latest
```

The API is now live at `http://localhost:8000`

## Useful Docker Commands

```bash
# List all images
docker images

# List running containers
docker ps

# Stop a running container
docker stop <container_id>

# Remove a container
docker rm <container_id>

# Remove the image
docker rmi iris-ml-pipeline:latest

# Run in detached (background) mode
docker run -d -p 8000:8000 --name iris-api iris-ml-pipeline:latest

# View container logs
docker logs iris-api

# Stop and remove in one step
docker stop iris-api && docker rm iris-api
```
